## 3. 가격 모델 설계

### 3.1 과금 방식: 크레딧 충전제 (권장)

**선택 이유:**
1. 월 구독 대비 심리적 진입 장벽이 낮음 (1만원 미만부터 시작 가능)
2. 사용량에 비례한 과금 → 사용자 불만 최소화
3. 경쟁사 가제트도 코인 차감형 채택 중 (시장 검증된 모델)
4. 대용량 패키지에 할인을 적용하면 업셀링 유도 가능
5. 미사용 크레딧은 이탈 방지 역할 (매몰비용 효과)

### 3.2 통합 크레딧 시스템

블로그 글과 이미지를 **하나의 크레딧**으로 통합합니다.

| 사용 기능 | 크레딧 차감 | 실제 원가 | 비고 |
|-----------|------------|-----------|------|
| 블로그 글 1편 생성 | **3 크레딧** | 136원 | Sonnet 4 기반 고품질 |
| 블로그 이미지 1세트 (8장) | **2 크레딧** | 92원 | Haiku + FLUX |
| 글+이미지 세트 | 5 크레딧 | 228원 | 번들 사용 시 할인 없음 (이미 합리적) |

> 1 크레딧 실질 원가: 블로그 글 기준 ~45원, 이미지 기준 ~46원 → **평균 ~45원/크레딧**

### 3.3 충전 플랜 (3단계 티어 + 디코이)

| 플랜 | 크레딧 | 가격 | 크레딧당 가격 | 할인율 | 마진율 | 용도 |
|------|--------|------|--------------|--------|--------|------|

| **프로** | 250 크레딧 | **49,900원** | 200원 | 39% | 78% | 헤비 유저/에이전시 |

#### 가격 설계 근거

**라이트 (9,900원 / 30크레딧)**
- 진입 장벽: 1만원 미만 → "한번 써볼까" 심리 유도
- 블로그 글 10편 또는 이미지 15세트 해당
- 원가: 30 x 45원 = 1,350원 → **마진 8,550원 (86%)**

**스탠다드 (24,900원 / 100크레딧) — 앵커링 타겟**
- 블로그 글 33편 또는 이미지 50세트 해당
- 월 1~2편/일 블로거에게 1~2개월분
- 원가: 100 x 45원 = 4,500원 → **마진 20,400원 (82%)**
- 라이트 대비 25% 할인 → 합리적 선택으로 유도

**프로 (49,900원 / 250크레딧) — 앵커링 역할**
- 블로그 글 83편 또는 이미지 125세트 해당
- 에이전시/다량 생산자 타겟
- 원가: 250 x 45원 = 11,250원 → **마진 38,650원 (78%)**
- 고가 플랜이 앵커로 작용 → 스탠다드 선택 유도

- 인스타 릴스: "30초 만에 블로그 글 완성" 시연 영상

#### 3. 네이버 카페/커뮤니티
- 소상공인 카페, 블로그 마케팅 커뮤니티, 1인 창업 카페
- "블로그 글 쓰는 데 3시간 vs 3분" 비교 후기
- 실제 상위노출 사례 공유

#### 4. 크몽/숨고 (마케팅 프리랜서 대상)
- 블로그 대행 프리랜서들이 뚝딱툴을 도구로 활용
- 프로 플랜 타겟팅: "월 83편 블로그 글, 49,900원이면 가능"
- 마케팅 대행사 B2B 세일즈

#### 5. Google Ads (검색 광고)
- 키워드: "AI 블로그 글쓰기", "블로그 글 자동 생성기", "블로그 이미지 만들기"
- 랜딩: 무료 체험 → 크레딧 충전 전환 퍼널
- 초기 월 30~50만원 예산으로 CPA 테스트

## 6. 번들 전략

### 6.1 글+이미지 워크플로우 연계

뚝딱툴의 핵심 차별점: **블로그 글 생성 → 이미지 자동 생성** 워크플로우.

이미 blog-image.js의 Parse 모드가 블로그 글을 분석하여 마커 기반 이미지를 생성하는 구조입니다.
이를 마케팅 메시지로 활용:

> "블로그 글 쓰고, 이미지까지 한 번에. 크레딧 5개로 완성."

### 6.2 번들 할인 제안 (향후 검토)

현재는 개별 크레딧 차감이 단순하고 명확하므로, 별도 번들 할인 없이 운영합니다.
사용자가 자연스럽게 글(3크레딧) + 이미지(2크레딧) = 5크레딧을 묶어 사용하게 됩니다.

향후 "블로그 올인원 세트" (글 1편 + 이미지 1세트 = 4크레딧)로 번들 할인을 테스트할 수 있습니다.

## 7. 리스크 & 대응

### 7.1 리스크 분석

| 리스크 | 심각도 | 가능성 | 대응 |
|--------|--------|--------|------|
| 뤼튼 등 무료 서비스와 경쟁 | 높음 | 높음 | 블로그 특화 차별화 + 품질 우위 강조 |
| 유료화 후 사용자 이탈 | 중간 | 높음 | 점진적 전환 + 무료 체험 유지 |
| API 비용 상승 | 중간 | 중간 | 마진율 78~86%로 여유 확보 |
| 충전만 하고 미사용 | 낮음 | 중간 | 크레딧 유효기간 6개월 설정 |
| 무료 한도만 사용하는 사용자 대부분 | 높음 | 높음 | 퍼널 최적화, 한도 도달 시 CTA 강화 |

### 7.2 경쟁 차별화 포인트

뤼튼 등 범용 AI 도구 대비 뚝딱툴의 차별점:
1. **네이버 블로그 특화**: 홈피드/SEO 최적화 시스템 프롬프트 (경쟁사에 없음)
2. **글+이미지 연계**: 글에서 자동으로 이미지 마커 파싱 → 이미지 생성
3. **한국 블로그 문화 이해**: 4가지 톤, 3가지 SEO 유형
4. **간편함**: 복잡한 프롬프트 불필요, 키워드만 입력
5. **가격 경쟁력**: 미드저니 월 $10~$60 vs 뚝딱툴 이미지 1세트 ~500원

## 10. 최종 권장안 요약

### 가격표

```
[라이트]  30크레딧  =  9,900원  (330원/크레딧)
[스탠다드] 100크레딧 = 24,900원  (249원/크레딧, 25% 할인)  ← 권장
[프로]    250크레딧 = 49,900원  (200원/크레딧, 39% 할인)
```

### 크레딧 사용

```
블로그 글 1편 생성    = 3 크레딧
블로그 이미지 1세트    = 2 크레딧
(글+이미지 세트 사용)  = 5 크레딧
```

### 무료 한도

```
블로그 글:    1회/일 (유료화 전 3회 → 1회로 축소)
블로그 이미지: 1크레딧/일 (유료화 전 3크레딧 → 1크레딧으로 축소)
스레드 글:    5회/일 (무료 유지)
제목/후킹:   무제한 (무료 유지)
```

### 월 300만원 달성 조건

```
필요 유료 사용자: ~175명
필요 MAU (전환율 3%): ~5,800명
필요 MAU (전환율 5%): ~3,500명
```

### 마케팅 우선순위

```
1순위: 네이버 블로그 SEO (자사 블로그 + 칼럼)
2순위: 스레드/인스타 바이럴 (무료 생성기가 마케팅 도구)
3순위: 소상공인 커뮤니티 (카페, 아이보스 등)
4순위: Google Ads (CPA 테스트)
5순위: 크몽/숨고 프리랜서 대상
```
