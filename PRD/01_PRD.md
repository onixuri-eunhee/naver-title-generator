## 1. 제품 개요

### 한 줄 요약
키워드 하나로 "조회수 터지는 숏폼"을 자동으로 벤치마킹하고, 그 공식대로 대본-영상까지 완성하는 AI 숏폼 제작 도구.

### 해결하는 문제
블로그 마케터/1인 사업자는 숏폼 영상이 필요하지만, (1) 어떤 구조가 터지는지 모르고, (2) 영상 제작 기술이 없고, (3) 외주 비용이 부담됨. 현재 v1은 바로 대본을 생성해서 품질이 들쭉날쭉함.

### 핵심 가치
- **벤치마킹 기반**: 실제 터진 숏폼을 먼저 분석하고 그 공식대로 만듦 (경쟁사 없음)
- **완전 자동**: 키워드 입력 → (벤치마킹~검수 전부 백그라운드) → 완성 영상 출력
- **한국 시장 최초**: 블로그 마케터 대상 숏폼 자동화 도구 부재 (리서치 확인)

---

## 2. 사용자

### 주요 사용자
- **누구**: 블로그 마케터, 1인 사업자, 소상공인 (현재 61명 가입)
- **상황**: 네이버 블로그로 마케팅 중인데, 숏폼(릴스/쇼츠)도 해야 한다는 압박
- **목표**: 블로그 글 주제로 숏폼 영상을 빠르게 만들어 SNS에 올리기

### 사용자 시나리오
1. 웨딩플래너가 "중학생 진로상담" 키워드를 입력하면
2. AI가 YouTube에서 조회수 터진 진로 숏폼 3~5개를 찾아 구조를 분석하고
3. 그 공식대로 대본-이미지-영상을 자동 생성해서 완성 MP4를 내려줌
4. 사용자는 다운받아 인스타 릴스에 업로드

## 4. 사용자 흐름 (User Flow)

### 핵심 흐름 (사용자 관점)
```
키워드 입력 → 로딩 (1~2분) → 완성된 대본+영상 확인 → 다운로드
```

### 백그라운드 파이프라인 (시스템 관점)
```
[Input] 키워드/블로그글
    │
    ▼
[Phase 1] 벤치마킹 ─── YouTube API 검색 → 조회수/구독자 비율 필터
    │                    → 자막 추출 → AI 구조 분석 → 패턴 구조화
    ▼
[Phase 2] 전략 설계 ─── 패턴 + 후킹 공식 10종 → 훅/인트로 자동 구성
    │                    → CTR 설계 (롱폼용 예약, 숏폼에선 스킵)
    ▼
[Phase 3] 기획서 ────── 대본 내용 설계 + 씬별 비주얼 프롬프트 마커
    │                    → 프리미엄 이미지생성기 프롬프트 로직 재활용
    ▼
[Phase 4] 대본+검수 ── 기획서 기반 Claude Sonnet 대본 생성
    │                    → AI 셀프 검수 (후킹/도입부/몰입력/첫화면)
    │                    → 불합격 시 자동 재생성 (최대 2회)
    ▼
[Phase 5] 생성 ──────── Supertone TTS → STT 워드 타이밍
    │                    → FLUX Schnell 이미지 (병렬)
    │                    → Kling 3.0 I2V (비디오 슬롯)
    │                    → Remotion 렌더링
    ▼
[Output] MP4 (1080×1920) + 대본 텍스트
```

## 5. 성공 기준

- [ ] 키워드 입력 후 2분 이내 완성 영상 출력
- [ ] 대본 후킹 품질 검수 80점 이상 통과율 70%+
- [ ] 30초 숏폼 원가 $0.50 이하
- [ ] B-roll 이미지 전체 씬 생성 성공률 95%+
- [ ] 사용자 만족도 조사 4점/5점 이상

## 6. 안 만드는 것 (Out of Scope)

> Phase 1에서 만들지 않습니다.

- AI 아바타 (HeyGen/D-ID) -- 이유: 30초당 $0.75~4.50 추가 비용으로 과금 체계 초과
- 인물 일관성 (LoRA 학습) -- 이유: 복잡도 높고 객체 중심 B-roll이 더 효과적
- SNS 자동 포스팅 -- 이유: Phase 2에서 추가
- 롱폼 영상 (5분+) -- 이유: Phase 3에서 추가
- 썸네일 자동 생성 -- 이유: 숏폼에서 불필요, Phase 3 롱폼 시 활성화
- 사용자 음성 복제 -- 이유: Phase 2에서 Supertone Clone 연동
