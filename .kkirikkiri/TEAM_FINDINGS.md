## 유형 A: 네이버 홈피드 노출용

> 네이버 앱 홈(MY) 피드의 AI 추천 알고리즘에 의해 노출되는 글.
> 검색 키워드보다 "클릭 유도력"과 "체류 시간"이 핵심.
> 블로그 지수와 무관하게 초보자도 노출 가능.

### 제목 작성 원칙
- **글자수**: 20~32자 (공백 포함). 모바일 피드에서 잘리지 않는 범위
- **키워드 위치**: 키워드를 앞에 두되, 감성적 후킹이 더 중요
- **제목 패턴**: 감성 자극형이 핵심
  - 경험 후기형: "3개월 써보니 진짜였던 OO" / "직접 해봤더니 달라진 것"
  - 반전형: "OO 안 하면 손해라고 느낀 순간"
  - 호기심 자극형: "이걸 모르고 OO한 사람이 아직도 있다고?"
  - 공감형: "OO하는 사장님들 다 공감할 이야기"
- **금지**: "안녕하세요", "오늘은 ~에 대해" 같은 진부한 시작

### 첫 문장/도입부 원칙
- 첫 100~150자 안에 핵심 메시지가 들어가야 함
- 독자의 상황/고민/감정에서 시작 (가게 소개 금지)
- 스크롤을 유도하는 긴장감/궁금증 생성
- 예시: "솔직히 말하면, 저도 처음엔 의심했습니다."

### 소제목 구조
- 이모지 + 짧은 한 줄 소제목 (정형화된 번호 매기기보다 자유로운 형태)
- 소제목 자체가 클릭 이후 스크롤을 유도하는 미니 후킹
- 3~4개 소제목 구성

### 키워드 배치 규칙
- 홈피드는 키워드 매칭보다 주제 관련성 평가
- 업종+지역 키워드를 자연스럽게 1~2회 배치
- 과도한 키워드 반복 금지 (AI가 스팸으로 인식)
- 자연어로 녹여내기 (키워드 나열 금지)

### 글 길이 권장
- 1000~1500자 (홈피드는 가독성과 체류시간 중심. 너무 길면 이탈)
- 문단은 1~3줄로 짧게 끊기
- 여백을 충분히 활용

### 해시태그/메타 전략
- 해시태그 5~8개 (핵심 주제 + 감성 태그 혼합)
- 예: #미용실일상 #머리스타일추천 #오늘의헤어 #동네미용실 #사장님일기
- 메타디스크립션: 감성적 한 줄 요약

## 유형 B: 네이버 SEO 최적화 (검색 상위노출)

> 네이버 검색(VIEW 탭)에서 특정 키워드로 상위 노출되는 글.
> C-Rank(전문성)와 D.I.A(독창성, 의도 일치) 로직 기반.
> 키워드 정확도 + 검색 의도 매칭이 핵심.

### 제목 작성 원칙
- **글자수**: 20~25자 (공백 포함). 네이버는 짧고 정확한 제목 선호
- **키워드 위치**: 타겟 키워드를 제목 맨 앞에 배치 (가장 중요)
- **제목 패턴**: [핵심 키워드 + 후킹 문장]
  - 숫자형: "강남역 미용실 추천 BEST 5 (직접 방문)"
  - 방법 제시형: "홈카페 라떼 만들기, 바리스타가 알려주는 3단계"
  - 비교형: "필라테스 vs 요가, 어떤 게 나한테 맞을까?"
  - 손실회피형: "네일 관리 이것만 안 해도 2주 더 유지됩니다"
- **중요**: 키워드를 띄어쓰기 포함 정확하게 작성

### 첫 문장/도입부 원칙
- 첫 문장에 핵심 키워드 자연스럽게 포함
- 검색자의 의도(니즈)에 직접 응답하는 형태
- "이 글에서 OO에 대해 알려드릴게요" 형태 금지
- 예시: "홍대 카페를 찾고 계신다면, 이 5곳은 꼭 가보세요."

### 소제목 구조
- 【01.】 【02.】 【03.】 형태의 번호형 소제목
- 소제목에도 세부 키워드 포함
- 3~5개 소제목 구성
- 구조화된 정보 전달 (네이버 D.I.A 로직이 구조화된 글 선호)

### 키워드 배치 규칙
- 1500~2000자 기준 메인 키워드 5~6회 배치
- 제목(1회) + 도입부(1회) + 소제목(2~3회) + 본문(1~2회) + 마무리(1회)
- 유사 키워드/연관 키워드를 함께 사용 (LSI 키워드)
- 키워드 밀도 0.8~1.5% 유지
- 키워드 스터핑(과도한 반복) 절대 금지

### 글 길이 권장
- 1500~2000자 (네이버 상위노출 최적 길이)
- 문단은 2~4줄로 구성
- 이미지 설명 위치에 "(이미지: OO 사진)" 형태로 이미지 삽입 가이드 표기

### 해시태그/메타 전략
- 해시태그 8~10개
- 구성: 메인 키워드(1~2개) + 세부 키워드(3~4개) + 지역 키워드(1~2개) + 일반 태그(2~3개)
- 예: #강남미용실 #강남역헤어 #레이어드컷 #여자머리추천 #강남맛집 #강남데이트 #헤어스타일 #미용실추천 #봄머리 #단발커트

## 유형 C: 구글 SEO 최적화

> 구글 검색에서 특정 키워드로 상위 노출되는 글.
> E-E-A-T(경험, 전문성, 권위, 신뢰) 기반 평가.
> 콘텐츠 깊이 + 구조화 + 사용자 의도 매칭이 핵심.

### 제목 작성 원칙
- **글자수**: 25~30자 (한글 기준. 30자 초과 시 구글이 잘라냄)
- **키워드 위치**: 타겟 키워드를 제목 앞부분에 배치
- **제목 패턴**: 정보성 + 구체성이 핵심
  - How-to형: "필라테스 초보 시작 가이드: 첫 수업 전 알아야 할 5가지"
  - 리스트형: "2026년 웨딩 컨설팅 선택 시 꼭 확인할 체크리스트 7"
  - 질문 응답형: "공부방 창업 비용, 실제로 얼마나 들까?"
  - 완벽 가이드형: "네일샵 인테리어 완벽 가이드 [비용부터 시공까지]"
- **중요**: 제목이 곧 검색 결과 화면의 title 태그 역할

### 첫 문장/도입부 원칙
- 첫 문단에서 검색 의도에 대한 직접 답변 제공
- 구글은 첫 150자를 메타 디스크립션으로 자동 추출하는 경우가 많음
- 정의형 문장으로 시작하면 AI 요약(SGE)에 인용될 확률 증가
- 예시: "카페 창업 비용은 평균 3,000~5,000만원 수준입니다. 규모와 지역에 따라..."

### 소제목 구조
- H2/H3 계층 구조 (SEO에서 매우 중요)
- ## (H2) 대주제 3~5개 + ### (H3) 세부 항목
- 소제목에 롱테일 키워드 포함
- FAQ 섹션 추가 권장 (구글 FAQ 리치 스니펫 획득 가능)

### 키워드 배치 규칙
- 키워드 밀도 0.8~1.2% (1000자당 8~12회)
- 제목(1회) + 첫 문단(1회) + H2 소제목(2~3회) + 본문(자연스럽게 분산) + 마무리(1회)
- 동의어/관련어를 풍부하게 사용 (구글의 시맨틱 검색에 유리)
- 자연스러운 문장 흐름 안에서만 사용 (키워드 스터핑은 구글이 가장 싫어하는 것)

### 글 길이 권장
- 2000~3000자 (구글은 깊이 있는 콘텐츠 선호)
- 문단은 3~5줄로 적당한 밀도
- 목차(Table of Contents) 포함 권장

### 해시태그/메타 전략
- 구글에서 해시태그는 SEO 효과 미미 → 대신 메타 태그에 집중
- meta description: 120~155자, 타겟 키워드 포함, 클릭을 유도하는 한 문장
- 구조화 데이터(Schema) 마크업 권장
- FAQ, HowTo 스키마가 특히 효과적

## 프롬프트 A: 네이버 홈피드 노출용

```
당신은 네이버 홈피드(홈판) 노출에 최적화된 블로그 글을 작성하는 전문 작가입니다.
실제 자영업자가 직접 쓴 것처럼 자연스럽고, 독자가 스크롤을 멈추고 읽게 만드는 글을 씁니다.

【글쓰기 스타일 규칙 - 반드시 지킬 것】
1. 문단은 반드시 1~3줄로 짧게 끊을 것. 한 문단이 4줄을 넘기면 안 됩니다.
2. 줄바꿈(\n\n)을 의도적으로 자주 사용해 호흡을 만들 것.
3. 도입부는 반드시 독자의 상황, 고민, 감정에서 시작할 것. 가게 소개로 시작 금지.
4. 강조하고 싶은 문장은 단독 줄로 분리해서 임팩트를 줄 것.
5. 전체 톤은 실제 사장님이 저녁에 하루를 정리하며 쓰는 일기처럼 자연스럽게.

【홈피드 노출 최적화 규칙】
1. 제목: 20~32자. 감성 자극형 또는 경험 후기형으로 작성.
   - 좋은 예: "3년째 이 방법 쓰는데 진짜 달라요", "이걸 모르고 00한 분들 꼭 보세요"
   - 나쁜 예: "00 맛집 추천 BEST 5", "안녕하세요 00입니다"
2. 첫 100자 안에 글의 핵심 메시지/감정이 드러나야 합니다.
3. 소제목: 이모지 + 짧은 한 줄 형태. 3~4개 구성.
4. 글 길이: 1000~1500자. 홈피드는 가독성과 체류시간이 핵심이므로 과도하게 길지 않게.
5. 업종과 지역 키워드는 자연스럽게 1~2회만 포함 (키워드 나열 금지).
6. 해시태그: 5~8개. 핵심 주제 태그 + 감성 태그 혼합.

【절대 금지사항】
- "안녕하세요", "오늘은 ~에 대해 알아보겠습니다", "~입니다" 반복
- 광고처럼 보이는 표현: 최고, 최저가, 1등, 강추, 대박, 완전 추천
- AI가 쓴 느낌이 나는 표현: "다양한", "특별한", "소중한", "여러분"
- 이모지 과다 사용 (소제목에만 하나씩)
- 동일 문장 구조 반복

【출력 형식 - 반드시 아래 JSON으로만 응답】
{
  "title": "제목 (20~32자, 감성 자극형 또는 경험 후기형)",
  "hook": "도입부 (독자 공감에서 시작. \\n\\n으로 문단 구분. 4~6문장. 짧고 감성적으로)",
  "body": "본문 (이모지+소제목 3~4개로 구성. 각 파트 짧은 문단. \\n\\n으로 구분. 전체 600~900자)",
  "cta": "마무리 (2~3문장. 자연스러운 공감 마무리. 방문 유도 느낌 없이)",
  "tags": ["태그1","태그2","태그3","태그4","태그5","태그6","태그7","태그8"]
}
```

## 프롬프트 B: 네이버 SEO 최적화 (검색 상위노출)

```
당신은 네이버 검색(VIEW 탭) 상위노출에 최적화된 블로그 글을 작성하는 SEO 전문 작가입니다.
네이버의 C-Rank(전문성)와 D.I.A(독창성, 의도일치, 충실성) 로직을 이해하고, 검색자의 의도에 정확히 부합하는 구조화된 글을 씁니다.

【글쓰기 스타일 규칙 - 반드시 지킬 것】
1. 문단은 2~4줄로 구성. 정보 밀도를 유지하되 읽기 쉽게.
2. 줄바꿈(\n\n)으로 문단을 명확히 구분할 것.
3. 도입부에서 검색 키워드와 관련된 독자의 니즈에 직접 응답할 것.
4. 소제목은 【01.】 【02.】 【03.】 형태로 번호를 매겨 구조화할 것.
5. 각 소제목 파트에서 구체적인 정보, 팁, 사례를 포함할 것.
6. 강조 문장은 단독 줄로 분리.

【네이버 SEO 최적화 규칙】
1. 제목: 20~25자. [핵심 키워드 + 후킹 문장] 구조. 키워드를 제목 맨 앞에 배치.
   - 좋은 예: "강남 미용실 추천 5곳, 직접 가본 후기", "필라테스 효과, 2개월 다닌 변화"
```

# 3. 톤 옵션 4종 설계

> 사용자가 선택할 수 있는 톤 4가지.
> 각 톤은 위 System 프롬프트의 `{tone}` 변수에 삽입되어 톤 가이드라인을 추가합니다.

---

## 톤 1: 친근한 구어체

```
【톤 가이드라인: 친근한 구어체】
- 말투: "~거든요", "~했어요", "~인 거죠", "~더라고요" 등 실제 대화체
- 문장 길이: 짧게. 한 문장에 하나의 생각만.
- 특징: 읽는 사람이 친구한테 듣는 느낌. 편안하고 부담 없이.
- 예시 문장:
  "이거 진짜 해보면 달라요."
  "저도 처음엔 몰랐거든요."
  "근데 막상 해보니까, 생각보다 쉽더라고요."
- 금지: 존댓말 과잉("-습니다" 연속), 딱딱한 설명체, 교과서 톤
```

## 톤 2: 전문가 톤

```
【톤 가이드라인: 전문가 톤】
- 말투: "~합니다", "~됩니다", "~기 때문입니다" 등 신뢰감 있는 서술체
- 문장 길이: 중간. 근거와 설명을 포함하되 간결하게.
- 특징: 이 분야에서 오래 일한 사람이 조언하는 느낌. 권위 있지만 거만하지 않게.
- 예시 문장:
  "실제로 현장에서 10년간 겪어본 결과, 이 방식이 가장 효율적입니다."
  "많은 분들이 간과하는 부분이 있는데, 바로 이것입니다."
  "경험상 이 단계를 건너뛰면 결과가 크게 달라집니다."
- 금지: 과도한 학술체, 지나친 전문 용어, 읽기 어려운 긴 문장
```

## 톤 3: 스토리텔링

```
【톤 가이드라인: 스토리텔링】
- 말투: "~했습니다", "~었어요", "그날", "그때" 등 이야기체
- 문장 길이: 리듬감 있게. 짧은 문장과 긴 문장을 교차.
- 특징: 하나의 에피소드를 중심으로 전개. 시간 순서 또는 감정 변화 흐름.
- 예시 문장:
  "그날 아침, 가게 문을 열자마자 전화가 울렸습니다."
  "손님이 돌아가시면서 한마디 하셨어요."
  "그 한마디가 하루 종일 머릿속을 맴돌았습니다."
- 금지: 사건 없이 설명만 나열, 감정 과잉(울먹울먹), 소설체 과도 사용
```

## 톤 4: 간결 실용체

```
【톤 가이드라인: 간결 실용체】
- 말투: "~입니다", "~하세요", "~됩니다" 등 깔끔한 정보 전달체
- 문장 길이: 최대한 짧게. 불필요한 수식어 제거. 핵심만.
- 특징: 바쁜 사람이 빠르게 훑어봐도 정보를 얻을 수 있게. 실용 정보 중심.
- 예시 문장:
  "준비물은 3가지입니다."
  "비용은 평균 30만원 선입니다."
  "핵심은 타이밍입니다. 너무 늦으면 효과가 반감됩니다."
- 금지: 감성적 표현, 장황한 배경 설명, 이모티콘/느낌표 남발
```

# 4. 제목/첫문장 원칙

---

## 4-1. 상위노출 제목 패턴 (8가지)

### 패턴 1: 손실회피형
- 원리: 사람은 이득보다 손실을 2배 더 크게 느낌 (프로스펙트 이론)
- 구조: "OO 안 하면 손해인 이유" / "이것 모르면 OO 망합니다"
- 예시:
  - "미용실 예약 전 이것 안 물어보면 후회합니다"
  - "카페 창업 이 실수만 안 해도 500만원 아낍니다"
  - "필라테스 이 동작 빼면 효과 반토막 납니다"

### 패턴 2: 역설형/반전형
- 원리: 상식을 뒤집어 호기심 자극
- 구조: "OO가 사실은 OO한 이유" / "OO 하지 마세요"
- 예시:
  - "맛있는 커피를 내리려면 원두를 아끼세요"
  - "손님이 많은 미용실이 오히려 위험한 이유"
  - "운동 열심히 하는데 살이 안 빠지는 진짜 이유"

### 패턴 3: 궁금증/호기심형
- 원리: 정보 격차(Information Gap)가 클릭을 유도
- 구조: "OO 해본 적 있으세요?" / "왜 OO일까?"
- 예시:
  - "동네 카페가 10년을 버틴 비결이 뭘까요?"
  - "이 네일 디자인, 왜 매달 1위일까?"
  - "공부방 원장이 절대 말 안 하는 것"

### 패턴 4: 숫자형
- 원리: 숫자는 구체성과 한정성을 줘서 신뢰도 상승
- 구조: "OO 추천 N가지" / "N분 만에 OO하는 법"
- 예시:
  - "카페 인테리어 비용 줄이는 5가지 방법"
  - "웨딩 컨설팅 고를 때 확인할 체크리스트 7"
  - "필라테스 3개월 차에 달라지는 몸의 변화 4가지"

### 패턴 5: 경험/후기형
- 원리: 실제 경험은 E-E-A-T 신뢰도를 높이고 공감 유도
- 구조: "N년/N개월 해본 솔직 후기" / "직접 OO 해봤습니다"
- 예시:
  - "미용실 3곳 다녀본 솔직 비교 후기"
  - "카페 사장 5년차가 말하는 현실"
  - "필라테스 6개월, 몸이 진짜 바뀌었을까?"

### 패턴 6: 비교형
- 원리: 선택에 고민하는 독자를 정확히 타겟팅
- 구조: "A vs B, 뭐가 나을까?" / "A와 B 비교"
- 예시:
  - "젤네일 vs 드라이케어, 어떤 게 오래 갈까?"
  - "요가 vs 필라테스, 초보자에게 맞는 건?"
  - "프랜차이즈 카페 vs 개인 카페, 수익 차이는?"

### 패턴 7: 방법 제시형
- 원리: "어떻게?"에 대한 직접적 답변 약속
- 구조: "OO하는 법" / "OO 단계별 가이드"
- 예시:
  - "미용실 단골 고객 만드는 3단계 방법"
  - "카페 메뉴판 만들 때 꼭 지켜야 할 원칙"
  - "네일샵 리뷰 관리하는 실전 방법"

### 패턴 8: 시의성/트렌드형
- 원리: 최신 정보에 대한 수요 타겟팅
- 구조: "2026년 OO 트렌드" / "올해 달라진 OO"
- 예시:
  - "2026년 봄 헤어 트렌드, 미용실 원장이 직접 추천"
  - "올해 웨딩 컨설팅 시장 달라진 3가지"
  - "2026 카페 인테리어 트렌드 총정리"

## 4-2. 네이버 vs 구글 제목 차이점

| 항목 | 네이버 | 구글 |
|------|--------|------|
| 최적 글자수 | 20~25자 | 25~30자 |
| 키워드 위치 | 제목 맨 앞 (필수) | 제목 앞부분 (권장) |
| 톤 | 친근 + 실용 | 정보성 + 구체성 |
| 후킹 요소 | 감성, 궁금증, 숫자 | 구체적 정보 약속, 범위 명시 |

| 예시 | "강남 미용실 추천 5곳 직접 방문 후기" | "강남 미용실 추천 가이드: 가격/실력 비교 [2026년]" |
| 피해야 할 것 | 제목에 블로그명 넣기 | 30자 초과 (잘림) |

## 4-3. 첫 문장으로 독자를 잡는 6가지 원칙

### 원칙 1: 독자 상황 공감으로 시작
- "머리를 바꾸고 싶은데 어디를 가야 할지 모르겠다면,"
- "카페를 열고 싶지만 비용이 걱정이라면,"

### 원칙 2: 질문으로 시작
- "혹시 매번 미용실에서 원하는 스타일이 안 나온 적 있으세요?"
- "카페 창업, 정말 3천만원이면 될까요?"

### 원칙 3: 놀라운 사실/수치로 시작

## 5-2. API 호출 시 프롬프트 조립 로직 (의사코드)

```javascript
// 1. 글 유형에 따라 system 프롬프트 선택
const systemPrompts = {
  'homefeed': SYSTEM_PROMPT_A,  // 네이버 홈피드용
  'naver-seo': SYSTEM_PROMPT_B, // 네이버 SEO용
  'google-seo': SYSTEM_PROMPT_C  // 구글 SEO용
};

// 2. 톤 가이드라인 선택
const toneGuides = {
  '친근한 구어체': TONE_1_GUIDE,
  '전문가 톤': TONE_2_GUIDE,
  '스토리텔링': TONE_3_GUIDE,
  '간결 실용체': TONE_4_GUIDE
};

// 3. system 프롬프트 조립
const system = systemPrompts[selectedType];

// 4. user 메시지 조립
const userMessage = `업종: ${industry}
${location ? '지역: ' + location : ''}
오늘의 소재: ${topic}
핵심 내용: ${memo}
톤: ${selectedTone}

${toneGuides[selectedTone]}`;

// 5. API 호출
fetch('/api/generate', {
  method: 'POST',
  body: JSON.stringify({
    model: 'claude-sonnet-4-20250514',
    max_tokens: 4096,
    system: system,
    messages: [{ role: 'user', content: userMessage }]
  })
});
```

## 5-3. 결과 렌더링 주의사항

- 구글 SEO 유형은 `meta_description`과 `faq` 필드가 추가로 있으므로 결과 UI에 해당 섹션 표시 필요
- 네이버 SEO 유형은 본문에 "(사진: OO)" 가이드가 포함되므로 별도 스타일링 권장
- JSON 파싱 시 `\n\n`을 실제 줄바꿈으로 변환하여 렌더링
- 해시태그 개수가 유형별로 다름 (홈피드 5~8개, 네이버SEO 8~10개, 구글 8개)
