## 1. 블로그 제목 생성기 (무료)

### 한줄 소개

업종과 키워드만 입력하면, 네이버 블로그 상위 노출에 최적화된 **클릭을 부르는 제목 12가지 패턴**을 자동으로 만들어드리는 도구입니다.

### 입력 필드 설명

| 필드 | 필수 여부 | 설명 |
|------|-----------|------|
| **업종 선택** (드롭다운) | 선택 | 웨딩/결혼, 요식업/카페, 온라인 강의/교육, 뷰티/피부/헤어, 부동산/인테리어, 헬스/운동, 마케팅/블로그, 기타 중 선택 |
| **핵심 키워드** (텍스트, 30자 이내) | 필수 | 블로그 글의 핵심 주제를 입력하세요 |

> 업종을 선택하면 해당 업종에 맞는 **추천 키워드 태그**가 아래에 표시됩니다.
> 클릭하면 키워드 칸에 자동으로 입력돼요.

### 좋은 입력 예시 vs 나쁜 입력 예시

| 구분 | 예시 | 이유 |
|------|------|------|
| 좋은 예 | "카페 창업 비용" | 구체적인 2~3단어 조합 (롱테일 키워드) |
| 좋은 예 | "강남 웨딩홀 가격 비교" | 지역 + 주제 + 의도가 명확 |
| 좋은 예 | "직장인 점심 다이어트" | 타겟 + 상황 + 주제가 담겨 있음 |
| 나쁜 예 | "카페" | 너무 광범위 — AI가 방향을 잡기 어려움 |
| 나쁜 예 | "맛있는 음식 만들기 비법 정리 모음" | 너무 길고 핵심이 불분명 |
| 나쁜 예 | "마케팅" | 한 단어만으로는 어떤 마케팅인지 알 수 없음 |

### 꿀팁 5개

1. **키워드는 2~4단어 조합이 최적입니다.** "블로그"보다 "자영업자 블로그 마케팅"처럼 구체적으로 입력할수록 경쟁이 낮고 검색 의도에 맞는 제목이 나옵니다.
2. **업종을 먼저 선택하세요.** 업종을 고르면 추천 키워드가 뜨는데, 이걸 클릭해서 시작하면 감을 잡기 쉽습니다.
3. **생성된 12가지 패턴 중 마음에 드는 2~3개를 골라 살짝 수정하세요.** "2개월 만에 매출 2배" 같은 표현은 실제 내용에 맞게 바꿔주는 게 좋습니다.
4. **같은 키워드로 여러 번 생성해 보세요.** AI가 매번 다른 제목을 만들어주기 때문에, 2~3번 돌려서 가장 좋은 것을 고르면 됩니다.
5. **핵심 키워드를 제목 맨 앞에 두세요.** 네이버는 앞쪽 단어에 더 높은 가중치를 줍니다. 생성된 제목에서 키워드 위치를 앞으로 옮겨보세요.

### 활용 시나리오

- **미용실 원장**: "손상모 펌" 키워드로 제목 12개 생성 → "손상모 펌, 진짜 해도 될까? 12년차 원장이 솔직하게 답합니다" 선택 → 내 경험에 맞게 수정
- **카페 사장**: "카페 시그니처 메뉴" 키워드로 생성 → 위협형 패턴에서 "이 메뉴 없으면 카페 매출 반토막? 시그니처 메뉴의 비밀" 선택
- **프리랜서 디자이너**: "포트폴리오 만들기" 키워드로 생성 → 호기심형 패턴에서 "포트폴리오 vs 경력서, 클라이언트가 진짜 보는 건?" 활용

### 자주 하는 실수

- **키워드를 너무 짧게 넣기**: "다이어트"만 넣으면 범위가 너무 넓어서 맞춤 제목이 나오기 어렵습니다.
- **생성된 제목을 수정 없이 그대로 쓰기**: 실제 글 내용과 맞지 않는 제목은 독자 이탈률을 높입니다. 내 글 내용에 맞게 꼭 다듬어주세요.
- **제목에 키워드를 여러 개 억지로 넣기**: 네이버 알고리즘은 키워드 도배를 저품질로 판단합니다. 핵심 키워드 1~2개만 자연스럽게 넣으세요.

### 결과 구성

생성 결과는 3가지 챕터, 총 **12가지 패턴**으로 나옵니다.

| 챕터 | 포함 패턴 | 성격 |
|------|-----------|------|
| Chapter 1. 긍정 메시지 | 이득+숫자형, 성공 사례형, 전문가 가이드형, 방법+체크리스트형 | "이걸 알면 이득" |
| Chapter 2. 위협 메시지 | 모르면 손해형, 공통점 경고형, 의심하라형, 공동의 적형 | "모르면 손해" |
| Chapter 3. 호기심 메시지 | 가치 입증형, 상식 파괴형, 질문&비교형, 타깃 호출형 | "궁금해서 클릭" |

> 각 제목 옆 [복사] 버튼으로 개별 복사, [전체 제목 복사] 버튼으로 12개 한번에 복사할 수 있습니다.

## 2. 후킹문구 생성기 (무료)

### 한줄 소개

업종과 키워드를 입력하면, 심리학 기반 **스크롤 멈추는 후킹문구 15개**를 즉시 생성하는 도구입니다. 인스타 릴스, 블로그 첫 줄, 유튜브 쇼츠, 전단지 헤드라인 등 어디서든 쓸 수 있습니다.

### 입력 필드 설명

| 필드 | 필수 여부 | 설명 |
|------|-----------|------|
| **업종** (텍스트, 30자 이내) | 필수 | 직접 입력하거나 아래 예시 태그 클릭. 어떤 업종이든 OK |
| **키워드 / 주제** (텍스트, 30자 이내) | 필수 | 홍보하고 싶은 내용을 자유롭게 입력하세요 |

> **업종 예시 태그**: 미용실, 필라테스, 카페, 식당, 학원, 부동산, 인테리어, 동물병원, 네일샵, 베이커리, 공방, 코칭
> **키워드 예시 태그**: 신규 고객, 단골 만들기, 매출 올리기, 다이어트, 탈모, 가성비, 리뷰 관리, 차별화, 초보자, SNS 마케팅

### 좋은 입력 예시 vs 나쁜 입력 예시

| 구분 | 업종 | 키워드 | 이유 |
|------|------|--------|------|
| 좋은 예 | "미용실" | "탈모 관리" | 업종과 키워드가 구체적으로 맞물림 |
| 좋은 예 | "온라인 쇼핑몰" | "첫 매출" | 상황이 명확해서 공감 문구가 잘 나옴 |
| 좋은 예 | "필라테스" | "체형 교정" | 서비스 특성이 분명 |
| 나쁜 예 | "사업" | "성공" | 너무 추상적 — 어떤 사업인지, 어떤 성공인지 모름 |
| 나쁜 예 | "" (빈칸) | "마케팅" | 업종 없이 키워드만 있으면 타겟이 불분명 |
| 나쁜 예 | "카페 베이커리 식당 꽃집" | "전부 다" | 한 번에 하나의 업종+키워드에 집중하세요 |

### 꿀팁 5개

1. **업종+키워드 조합을 바꿔가며 여러 번 생성하세요.** 같은 "미용실 + 탈모"라도 누를 때마다 100가지 이상의 풀에서 랜덤으로 15개가 선정됩니다.
2. **생성된 문구 뒤에 내 사업의 구체적 숫자를 붙이면 효과가 2배입니다.** 예: "10명 중 9명이 모르는 탈모 관리법" → "10명 중 9명이 모르는 탈모 관리법 (우리 매장 3개월 데이터)"
3. **인스타 릴스 첫 자막, 카드뉴스 첫 장에 그대로 넣으세요.** 후킹문구는 0.3초 안에 스크롤을 멈추게 하는 것이 목적이니까, 가장 눈에 띄는 첫 줄에 배치하세요.
4. **한글 조사(은/는, 이/가, 을/를)가 자동 처리됩니다.** "필라테스가" "필라테스를" 등이 자연스럽게 붙으니 그대로 사용하셔도 됩니다.
5. **마음에 드는 문구는 바로 [복사] 버튼 클릭!** 개별 복사 또는 [전체 복사]로 15개를 한번에 가져갈 수 있습니다.

### 활용 시나리오

- **네일샵 원장**: 업종 "네일샵" + 키워드 "여름 네일" → 생성된 문구를 인스타 릴스 첫 줄에 사용
- **1인 강사**: 업종 "코칭" + 키워드 "수강생 모집" → 카카오톡 채널 메시지 헤드라인으로 활용
- **부동산 중개사**: 업종 "부동산" + 키워드 "전세 사기 예방" → 블로그 글 첫 문장 후킹으로 사용
- **베이커리 사장님**: 업종 "베이커리" + 키워드 "신메뉴" → 네이버 플레이스 소식 작성 시 첫 줄로 활용

### 자주 하는 실수

- **업종 칸에 여러 업종을 한꺼번에 넣기**: "카페 식당 꽃집" 이렇게 넣으면 문구가 어색해집니다. 하나씩만 넣으세요.
- **키워드를 문장으로 넣기**: "우리 가게 매출을 올리고 싶어요" 대신 "매출 올리기"처럼 핵심 단어만 넣으세요.
- **생성된 문구를 제목으로 쓰기**: 후킹문구는 제목이 아니라 **첫 줄** 용도입니다. 블로그 제목이 필요하면 제목 생성기를 쓰세요.

### 결과 구성

14가지 심리 후킹 공식(패턴 인터럽트, 손실회피, 호기심 폭발, 구체성/수치, 정체성 호출, 사회적 증거, 문제공감, 상식 비틀기, 욕망 자극, 권위부여, 오픈루프, 즉시성, 비밀/은밀함, 비교 자극) 중에서 매번 랜덤으로 15개를 선별해서 보여줍니다.

## 3. 스레드 글 생성기 (무료)

### 한줄 소개

소재만 입력하면 **터지는 Threads(스레드) 글 3개**를 AI가 뚝딱 만들어주는 도구입니다. 글 유형과 말투를 선택할 수 있어서, 내 브랜드 느낌에 맞는 글이 나옵니다.

### 입력 필드 설명

| 필드 | 필수 여부 | 설명 |
|------|-----------|------|
| **글 유형** (버튼 선택) | 필수 | 정보형, 공감형, 반전형, 궁금증형 중 택 1 |
| **말투** (버튼 선택) | 필수 | 친구체, 해요체, 단문체, 격식체 중 택 1 |
| **업종/직종** (텍스트) | 선택 | 예: 카페, 강사, 마케터 |
| **타겟 독자** (텍스트) | 선택 | 예: 30대 직장인, 초보 창업자 |
| **주제/소재** (텍스트) | 필수 | 예: 번아웃 극복, 단골 만들기, 첫 매출 |
| **내 경험이나 메모** (텍스트, 여러 줄) | 선택 | 짧게 적어도 됩니다. 글의 방향이 더 정확해져요 |

> [랜덤] 버튼을 누르면 예시 소재가 자동으로 채워집니다. 감 잡기에 좋아요.

### 글 유형 상세 설명

| 유형 | 특징 | 추천 상황 |
|------|------|-----------|
| **정보형** | "N가지 팁" 숫자 리스트로 가치 전달 | 전문 지식 공유, 노하우 전달 |
| **공감형** | 감정을 짚고 경험담으로 위로 | 고객 고충 공감, 신뢰 구축 |
| **반전형** | 상식을 뒤엎는 한 방으로 호기심 유발 | 화제성 콘텐츠, 의견 표출 |
| **궁금증형** | 스토리 절반만 공개, 댓글 폭발 | 참여 유도, 팔로워 늘리기 |

### 말투 상세 설명

| 말투 | 느낌 | 예시 |
|------|------|------|
| **친구체** | ~했어, ~이야 (편한 반말) | "이거 진짜 대박이야" |
| **해요체** | ~해요 (따뜻하고 부드럽게) | "한번 해보세요, 달라져요" |
| **단문체** | 짧고 건조. SNS 특유 리듬 | "멈췄다. 생각했다. 바꿨다." |
| **격식체** | ~합니다 (전문가 느낌) | "3가지 핵심을 정리하겠습니다" |

### 좋은 입력 예시 vs 나쁜 입력 예시

| 구분 | 주제/소재 | 메모 | 이유 |
|------|-----------|------|------|
| 좋은 예 | "퇴근 후 부업 첫 달" | "처음엔 월 10만원도 기뻤음" | 구체적 상황 + 감정이 담김 |
| 좋은 예 | "단골 만드는 법" | "커피보다 사람이 오는 가게" | 핵심 메시지가 명확 |
| 나쁜 예 | "성공" | (빈칸) | 너무 추상적, 메모도 없음 |
| 나쁜 예 | "여러 가지 이야기" | "그냥 생각나는 거 다" | 방향이 없으면 AI도 방향을 못 잡음 |

### 꿀팁 5개

1. **"메모" 칸을 적극 활용하세요.** 한 줄이라도 내 경험을 적으면 AI가 그 방향으로 글을 써줍니다. "처음엔 반신반의했는데 한 달 뒤 달라짐" 이 한 줄만으로도 글 퀄리티가 확 올라갑니다.
2. **결과는 3개(안 1, 안 2, 안 3)가 나옵니다.** 탭을 눌러 비교해보고 가장 마음에 드는 것을 고르세요.
3. **업종/직종과 타겟 독자를 채우면 맞춤도가 올라갑니다.** 이 두 칸은 선택이지만, 채우면 "누가 누구에게 하는 말인지"가 명확해져서 글이 훨씬 자연스러워집니다.
4. **[랜덤] 버튼으로 영감을 받으세요.** 소재가 안 떠오를 때 랜덤을 눌러 예시를 보고, 내 상황에 맞게 수정하면 됩니다.
5. **단문체 + 반전형 조합은 스레드에서 바이럴되기 가장 좋은 조합입니다.** 짧은 호흡 + 상식 뒤집기는 댓글과 공유를 부릅니다.

### 활용 시나리오

- **카페 사장**: 정보형 + 친구체 → "카페에서 단골 만드는 3가지 방법" 스레드 글 생성 → 바로 Threads에 게시
- **프리랜서 디자이너**: 공감형 + 해요체 → "프리랜서 2년 차, 번아웃 올 때 이렇게 극복했어요" → 타겟 독자와 공감대 형성
- **온라인 강사**: 궁금증형 + 단문체 → "수강생 0명에서 50명. 광고 한 번 안 썼다." → 결말이 궁금해서 팔로우 유도
- **마케터**: 반전형 + 격식체 → "인스타 팔로워보다 중요한 것이 있습니다" → 전문성 어필

### 자주 하는 실수

- **주제/소재를 너무 넓게 잡기**: "마케팅"보다 "인스타 팔로워보다 중요한 것" 처럼 하나의 관점으로 좁혀주세요.
- **메모 칸을 비워두기**: AI가 일반적인 글을 쓰게 됩니다. 내 경험이나 에피소드를 한 줄이라도 넣어야 "나만의 글"이 됩니다.
- **생성된 글을 수정 없이 그대로 올리기**: 글 끝에 내 계정에 맞는 CTA(댓글 유도 질문 등)를 추가하면 반응이 더 좋아집니다.

### 결과 구성

- 3가지 안(안 1, 안 2, 안 3) 탭으로 전환 가능
- 각 안마다 글자 수 표시
- [복사] 버튼으로 바로 복사 가능

## 4. 블로그 글 생성기 (PRO)

### 한줄 소개

업종과 주제만 입력하면 **네이버 홈피드, 네이버 SEO, 구글 SEO에 최적화된 상위노출 블로그 글**을 AI가 자동으로 작성해주는 도구입니다. 제목, 도입부, 본문, 해시태그, 내부링크 추천까지 한 번에 나옵니다.

### 입력 필드 설명

| 필드 | 필수 여부 | 설명 |
|------|-----------|------|
| **글 유형** (버튼 선택) | 필수 | 네이버 홈피드 / 네이버 SEO / 구글 SEO 중 택 1 |
| **톤 선택** (버튼 선택) | 필수 | 친근한 구어체 / 전문가 톤 / 스토리텔링 / 간결 실용체 중 택 1 |
| **마무리 CTA** (버튼 선택) | 필수 | 상담/예약 유도 / 방문 유도 / 공감 마무리 / 직접 입력 중 택 1 |
| **업종/입장** (텍스트, 30자 이내) | 필수 | 예: 미용실 원장, 카페 사장, 뷰티 블로거 |
| **주제/키워드** (텍스트, 30자 이내) | 필수 | 예: 손톱 관리법, 다이어트 식단 |
| **지역** (텍스트, 20자 이내) | 선택 | 예: 강남, 홍대, 부산 서면 |
| **추가 요청사항** (텍스트, 여러 줄) | 필수 | 나만의 경험, 느낌, 후기를 적어주세요 |

### 글 유형 상세 설명

| 유형 | 특징 | 추천 상황 |
|------|------|-----------|
| **네이버 홈피드** | 홈 탭 AI 추천 노출용. 체류시간과 클릭률 최적화 | 네이버 앱 홈 탭 노출을 노리는 경우 |
| **네이버 SEO** | 네이버 검색(VIEW) 상위노출. 설명문 포함 | 네이버 검색 결과 상단 노출이 목표 |
| **구글 SEO** | 구글 검색 상위노출. 메타 디스크립션+FAQ 포함 | 구글 유입을 늘리고 싶은 경우 |

### 톤 상세 설명

| 톤 | 느낌 | 추천 상황 |
|----|------|-----------|
| **친근한 구어체** | ~거든요, ~더라고요 (편한 대화체) | 자영업자 블로그, 후기성 글 |
| **전문가 톤** | ~합니다 (신뢰감 있는 서술체) | 전문 분야 글, 가이드형 콘텐츠 |
| **스토리텔링** | 에피소드 중심, 감정 변화 흐름 | 경험담, 창업 스토리, 성공/실패 사례 |
| **간결 실용체** | 핵심만, 불필요한 수식어 제거 | 리스트형 글, 비교 리뷰, 정보 전달 |

### CTA 상세 설명

| CTA | 설명 | 예시 |
|-----|------|------|
| **상담/예약 유도** | 카톡, 전화 문의 전환 | "카톡으로 상담 문의 주세요" |
| **방문 유도** | 오프라인 매장 방문 유도 | "근처 오시면 한번 들러보세요" |
| **공감 마무리** | 하드셀 없이 자연스럽게 | "도움이 되셨길 바랍니다" |
| **직접 입력** | 원하는 CTA 직접 작성 | "네이버 예약으로 10% 할인 받으세요" |

### 좋은 입력 예시 vs 나쁜 입력 예시

| 구분 | 업종/입장 | 주제/키워드 | 추가 요청사항 | 이유 |
|------|-----------|-------------|---------------|------|
| 좋은 예 | "미용실 원장" | "손상모 펌" | "12년 경력, 저온 디지털펌 전문, 산성 약제 사용, 처음엔 반신반의했는데 한 달 뒤 머릿결이 확 달라짐" | 입장이 명확하고, 경력+전문성+에피소드가 풍부 |
| 좋은 예 | "카페 사장" | "시그니처 메뉴 개발" | "통장 잔고 200만원까지 떨어졌을 때 단골 한 분이 살려줬음, 카라멜 당도 30% 낮추고 에스프레소 추가" | 스토리가 있고 구체적인 숫자가 포함됨 |
| 나쁜 예 | "사장" | "마케팅" | (빈칸) | 어떤 사장인지, 어떤 마케팅인지 모르고 경험이 없음 |
| 나쁜 예 | "블로거" | "좋은 글" | "잘 써주세요" | 주제가 추상적이고 AI가 참고할 정보가 없음 |

### 꿀팁 5개

1. **"추가 요청사항"이 글의 품질을 결정합니다.** 이 칸에 내 경험, 경력 년수, 구체적 에피소드, 손님 반응 등을 넣을수록 AI가 아닌 "나만의 글"이 됩니다. 많이 적을수록 좋습니다.
2. **"업종/입장"은 역할까지 구체적으로 쓰세요.** "미용실"이 아니라 "미용실 원장"으로, "카페"가 아니라 "카페 사장" 또는 "카페 방문 블로거"로 써야 글의 관점이 달라집니다.
3. **예시 캐러셀을 먼저 확인하세요.** 페이지 상단에 실제 생성 예시가 슬라이드로 있습니다. 홈피드/네이버SEO/구글SEO 별로 어떤 글이 나오는지 미리 확인하고 방향을 잡으세요.
4. **지역을 넣으면 지역 키워드가 자연스럽게 포함됩니다.** "강남 미용실", "홍대 카페" 같은 지역 키워드가 제목과 본문에 녹아들어 지역 검색 노출에 유리합니다.
5. **생성 후 "이 글에 맞는 이미지 생성하기" 버튼을 누르세요.** 블로그 글이 이미지 생성기로 자동 전달되어, 글 내용에 딱 맞는 이미지를 바로 만들 수 있습니다.

### 활용 시나리오

- **필라테스 강사**: 네이버 SEO + 전문가 톤 + 상담/예약 유도 → "체형 교정 필라테스" 주제 → 3개월 변화 과정 기록형 글 생성
- **맛집 블로거**: 네이버 SEO + 친근한 구어체 + 공감 마무리 → "강남 파스타 맛집" 주제 + 지역 "강남" → 비교 리뷰형 글 생성
- **인테리어 업체**: 구글 SEO + 전문가 톤 + 상담/예약 유도 → "소형 아파트 리모델링 비용" → FAQ 포함 종합 가이드 생성
- **네일샵 원장**: 네이버 홈피드 + 간결 실용체 + 방문 유도 → "여름 네일 트렌드" → 리스트형 트렌드 글 생성

### 자주 하는 실수

- **추가 요청사항을 비우거나 "잘 써주세요"만 쓰기**: AI가 일반적인 글을 쓰게 됩니다. 내 경력, 에피소드, 구체적 숫자를 넣어야 차별화됩니다.
- **글 유형을 고민 없이 선택하기**: 네이버 홈피드는 체류시간 최적화, 네이버 SEO는 검색 키워드 최적화, 구글 SEO는 FAQ+메타 디스크립션 포함. 목적에 맞게 선택하세요.
- **생성된 글을 읽어보지 않고 바로 발행하기**: 내용 중 내 상황과 맞지 않는 숫자나 사례가 있을 수 있습니다. 반드시 한 번 읽고 수정한 뒤 발행하세요.

### 결과 구성

| 결과 항목 | 설명 |
|-----------|------|
| **제목** | SEO 최적화된 블로그 제목 |
| **설명문** (네이버 SEO 전용) | 검색 결과에서 제목 아래 표시되는 설명문 |
| **메타 디스크립션** (구글 SEO 전용) | 구글 검색 결과에 노출되는 설명 |
| **도입부** | 독자를 사로잡는 후킹 첫 문단 |
| **본문** | 소제목+내용으로 구성된 메인 콘텐츠 (사진 마커 포함) |
| **FAQ** (구글 SEO 전용) | 자주 묻는 질문과 답변 |
| **마무리** | 선택한 CTA에 맞는 마무리 문단 |
| **해시태그** | 노출에 유리한 해시태그 모음 |
| **함께 보면 좋은 글** | 내부링크 추천 (URL 교체 가능) |

> 각 항목마다 [복사] 버튼이 있고, [전체 글 한 번에 복사]로 한번에 가져갈 수도 있습니다.
> AI 글 검수기가 자동으로 글의 품질을 점수로 평가하고 개선 제안을 해줍니다.

## 5. 블로그 이미지 생성기 (PRO)

### 한줄 소개

블로그 글을 붙여넣거나 주제를 입력하면, **문맥에 맞는 AI 이미지를 자동 생성**하고 **썸네일에 텍스트까지 합성**해주는 도구입니다.

### 입력 방식 (2가지 탭)

#### 탭 1: 글에서 생성 (추천)

| 필드 | 필수 여부 | 설명 |
|------|-----------|------|
| **블로그 글 붙여넣기** (텍스트, 여러 줄) | 필수 | 블로그 글 생성기에서 만든 글을 붙여넣으세요. (사진: ...) 마커가 포함된 글이면 각 위치에 맞는 이미지가 생성됩니다 |
| **이미지 마커 편집** (자동 감지) | 자동 | 글에서 (사진: ...) 마커가 감지되면 수정/삭제/추가 가능 (최대 10개) |
| **썸네일 텍스트** (텍스트, 30자 이내) | 필수 | 대표이미지에 합성될 텍스트. 예: "카페 창업 비용 총정리" |

#### 탭 2: 직접 입력

| 필드 | 필수 여부 | 설명 |
|------|-----------|------|
| **블로그 주제** (텍스트, 200자 이내) | 필수 | 예: 강남 카페 인테리어, 자영업자 브랜딩 전략 |
| **분위기 / 스타일** (드롭다운) | 필수 | 밝고 깔끔한 / 따뜻하고 아늑한 / 전문적인 / 감성적인 중 선택 |
| **썸네일 텍스트** (텍스트, 30자 이내) | 필수 | 대표이미지에 합성될 텍스트 |

### 좋은 입력 예시 vs 나쁜 입력 예시

| 구분 | 예시 | 이유 |
|------|------|------|
| 좋은 예 (글에서 생성) | 블로그 글 생성기에서 만든 글 전체 붙여넣기 | (사진: ...) 마커가 자동 감지되어 각 위치에 맞는 이미지 생성 |
| 좋은 예 (직접 입력) | "카페 인테리어, 밝은 조명과 우드톤 테이블, 따뜻한 분위기" | 구체적 장면이 묘사됨 |
| 좋은 예 (썸네일) | "카페 창업 비용 총정리" | 15자 내외로 임팩트 있게 |
| 나쁜 예 (직접 입력) | "좋은 이미지" | 어떤 이미지인지 AI가 알 수 없음 |
| 나쁜 예 (썸네일) | "" (빈칸) | 썸네일 텍스트는 필수입니다 |
| 나쁜 예 (썸네일) | "2026년 최신 강남역 근처 숨은 보석 같은 파스타 맛집 총정리 리뷰" | 너무 길면 썸네일에서 읽기 어려움 |

### 꿀팁 5개

1. **블로그 글 생성기와 연동해서 쓰세요.** 블로그 글 생성기 결과 페이지 하단의 [이 글에 맞는 이미지 생성하기] 버튼을 누르면, 글 내용이 자동으로 전달되어 마커가 감지됩니다.
2. **마커는 수정, 삭제, 추가가 가능합니다.** 자동 감지된 마커가 마음에 안 들면 내용을 수정하거나, 불필요한 건 삭제하고, 원하는 장면을 추가할 수 있습니다.
3. **썸네일 텍스트는 10~15자가 최적입니다.** 대표 이미지 위에 큰 글씨로 합성되기 때문에, 짧고 임팩트 있게 쓰는 게 좋습니다.
4. **직접 입력 모드는 글 없이도 사용 가능합니다.** 주제와 분위기만 선택하면 8장의 이미지가 생성됩니다. 블로그 글 없이 이미지만 필요할 때 유용합니다.
5. **마음에 안 들면 [전체 재생성] 버튼을 누르세요.** 1크레딧으로 새로운 이미지를 다시 생성할 수 있습니다. 강화된 프롬프트로 더 나은 결과가 나올 수 있어요.

### 활용 시나리오

- **미용실 원장**: 블로그 글 생성기에서 "손상모 펌" 글 작성 → [이미지 생성하기] 클릭 → 시술 전후 이미지, 약제 이미지 등이 자동 생성 → 다운로드해서 블로그에 첨부
- **인테리어 업체**: "소형 아파트 리모델링" 글의 (사진: 시공 전후 비교) 마커에 맞는 이미지 생성
- **카페 사장**: 직접 입력 모드 → "카페 시그니처 메뉴, 카라멜라떼, 따뜻한 분위기" → 분위기 "따뜻하고 아늑한" 선택 → 인스타/블로그용 이미지 8장 생성

### 자주 하는 실수

- **썸네일 텍스트를 너무 길게 쓰기**: 30자까지 가능하지만, 10~15자가 보기 좋습니다. 길면 이미지 위에서 읽기 어렵습니다.
- **마커 없는 글을 붙여넣고 "이미지가 안 만들어져요"**: (사진: ...) 또는 (이미지: ...) 마커가 없으면 이미지를 만들 수 없습니다. 마커를 직접 추가하거나, 블로그 글 생성기에서 만든 글을 사용하세요.
- **이미지를 생성만 하고 다운로드 안 하기**: 생성된 이미지는 영구 보관되지 않으니, 마음에 드는 이미지는 반드시 [다운로드] 버튼으로 저장하세요.

### 결과 구성

- 첫 번째 이미지: **썸네일** (텍스트가 합성된 대표 이미지)
- 나머지 이미지: 각 마커에 맞는 본문용 이미지
- 각 이미지마다 [다운로드] 버튼 제공
- [전체 재생성] 버튼으로 모든 이미지를 새로 생성 가능

## 6. 추천 워크플로우 -- 도구 조합 활용법

### 기본 워크플로우: 블로그 글 하나 완성하기

```
STEP 1. 블로그 제목 생성기 (무료)
  → 키워드 입력 → 12가지 패턴 중 최적 제목 선택

STEP 2. 후킹문구 생성기 (무료)
  → 같은 업종+키워드로 15개 문구 생성
  → 블로그 첫 문장에 활용할 후킹 선택

STEP 3. 블로그 글 생성기 (PRO)
  → STEP 1에서 고른 키워드와 STEP 2에서 고른 후킹 방향을 참고
  → 업종/입장 + 주제/키워드 + 추가 요청사항 입력
  → 제목, 도입부, 본문, 마무리, 해시태그 한 번에 생성

STEP 4. 블로그 이미지 생성기 (PRO)
  → STEP 3에서 생성된 글의 [이 글에 맞는 이미지 생성하기] 클릭
  → 글이 자동 전달 → 마커별 이미지 생성 → 다운로드

STEP 5. 네이버 블로그에 발행
  → 제목 + 본문 + 이미지 조합해서 발행
```

### 스레드 콘텐츠 워크플로우

```
STEP 1. 스레드 글 생성기 (무료)
  → 유형과 말투 선택 → 주제 입력 → 3가지 안 중 택 1 → 복사

STEP 2. 후킹문구 생성기 (무료)
  → 같은 주제로 문구 생성 → 스레드 글 첫 줄을 더 강하게 교체

STEP 3. Threads 앱에서 발행
```

### SNS 마케팅 집중 워크플로우

```
STEP 1. 후킹문구 생성기 (무료)
  → 이번 주 홍보할 내용으로 후킹문구 15개 생성
  → 월~금 하루 3개씩 인스타 릴스 첫 줄에 활용

STEP 2. 스레드 글 생성기 (무료)
  → 같은 주제로 스레드 글 3개 생성
  → 일주일에 3회 스레드 발행

STEP 3. 블로그 글 생성기 (PRO)
  → 주 1회 상세 블로그 글 작성
  → 스레드/인스타에서 블로그로 유입 유도
```

### 상황별 추천 도구

| 상황 | 추천 도구 | 이유 |
|------|-----------|------|
| "블로그 제목이 안 떠올라요" | 블로그 제목 생성기 | 12가지 패턴으로 즉시 해결 |
| "인스타 첫 줄 뭐라고 쓰지?" | 후킹문구 생성기 | 심리학 기반 15개 문구 |
| "스레드 뭐 올리지?" | 스레드 글 생성기 | 소재만 넣으면 글 3개 완성 |
| "블로그 글 쓸 시간이 없어요" | 블로그 글 생성기 | 10분 안에 전체 글 완성 |
| "블로그 이미지가 필요해요" | 블로그 이미지 생성기 | 글에 맞는 이미지 자동 생성 |
| "블로그 글 + 이미지 한번에" | 글 생성기 → 이미지 생성기 | 버튼 하나로 연동 |
