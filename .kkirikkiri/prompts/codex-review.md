# 코드 리뷰 요청: 황금키워드 기능

## 대상 파일
1. api/keywords.js - 서버사이드 키워드 분석 로직
2. keyword-finder.html - 프론트엔드 UI

## 리뷰 관점
1. **버그**: blogCount가 0으로 기본값 처리되어 포화도 점수가 부풀려지는 문제
2. **로직 오류**: calculateGoldenScore() 함수의 점수 배분 적절성
3. **프롬프트 품질**: Claude Haiku 시드 키워드 생성 프롬프트가 입력 분야와 무관한 키워드를 생성하는 문제
4. **UX 버그**: "이 키워드로 글쓰기" 버튼 클릭 시 blog-writer.html에 키워드가 전달되지 않는 문제
5. **API 효율성**: 네이버 API 호출 최적화

각 문제에 대해 원인과 수정안을 제시해주세요.
