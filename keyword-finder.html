입력 항목별 작성법

내 분야업종 또는 블로그 카테고리
예: 웨딩컨설팅, 카페 창업, 육아/리빙, 인테리어, 부동산
나는본인의 역할이나 직업
예: 웨딩플래너, 카페 사장, 육아 블로거, 인테리어 디자이너
타겟 독자구체적인 페르소나를 쓸수록 정확도 UP (200자)
예: 30대 초반 예비 신부. 결혼 준비가 처음이라 뭐부터 해야 할지 모르는 상태. 스드메, 웨딩홀, 예식 비용에 관심이 많고 합리적인 가격을 찾고 있다.
자주 받는 질문한 줄에 하나씩, 많이 쓸수록 시드키워드가 풍부해짐
예: 스드메 비용이 얼마예요? / 본식 드레스 어떻게 고르나요? / 셀프웨딩 가능한가요?
직접 추가할 키워드AI가 모를 수 있는 업계 전문 용어 직접 입력
예: 스드메, 본식드레스, 웨딩박람회, 신혼여행 (쉼표 또는 줄바꿈으로 구분)

프로필 저장/불러오기

  • 저장: 분야, 역할, 타겟을 입력한 뒤 [저장] 버튼 → 프로필 이름 입력 → 여러 개 저장 가능
  • 불러오기: 드롭다운에서 저장된 프로필 선택 → 자동으로 채워짐
  • 삭제: 프로필 선택 후 [삭제] 버튼
  • 여러 직업/분야를 가진 분은 프로필을 전환하면서 편하게 사용하세요

결과 활용법

  • 🔵 블루오션: 즉시 블로그 글을 쓸 가치가 있는 키워드
  • 🟢 틈새 공략: 충분히 도전할 만한 키워드
  • 포화도 3 이하: 경쟁 글이 적은 블루오션 — 빠르게 선점하세요
  • 트렌드 상승: 지금 쓰면 검색량이 더 올라갈 키워드
  • [글쓰기] 버튼: 클릭하면 해당 키워드로 블로그 글 생성기로 바로 이동
  • CSV 다운로드: 전체 키워드 목록을 엑셀에서 분석할 수 있습니다
